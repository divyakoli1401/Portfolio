All about Stylesheet

Let's Begin with What exactly is CSS? So CSS stands for Cascading Style Sheet. Now you must be thinking what is the use of this style sheet? Well as the name itself suggests, Style sheet in simple terms is used to apply style to your different html tags.

Different Types of Style Sheets:

  1. Inline Style Sheeet
  2. Internal Style Sheet
  3. External Style Sheet

As you can tell from the above list, Style Sheets are of three types.Each type has it's own set of advantages.So now let's find what's the difference between these types.

Internal Stylesheet

We will begin with Inline Style Sheet. Inline Style Sheet is used when you want to apply style to a particular html element. You declare Inline Style within individual tags and it shows it's styling effect only on those particular tags or elements. The style attribute is used to declare Inline Stylesheets.

Internal Stylesheet

The other type is Internal Style Sheet. Unlike Inline Stylesheet which is valid for only individual html element, Internal Stylesheet is used when you want a unique style to a particular HTML document. The style tag is used for declaring Internal Stylesheet. It is declared in the head section of the html document.

External Stylesheet

Now coming to the most widely used External Style Sheet. External Stylesheets are used when you want to apply the same style to multiple pages. If you make changes in the External Stylesheet those changes will be reflected in all the pages where the stylesheet is used. For using External stylsheet we need to make an external file with .css extension. Then we need to use link tag in the head section of HTML document, with attributes like href and rel.

Multiple Stylesheets

You can have multiple stylesheet definitions on one page. For instance, an Inline stylesheet and an External stylesheet on one page or an Inline stylesheet and an Internal stylesheet on one page, or all three together. If a property has been set for the same HTML element in different stylesheet definitions on the same HTML page, the definition that will be used will be from the stylesheet that has the highest priority. Stylesheets by priority:

  • Inline stylesheet - An Inline stylesheet has the highest priority. It will override styles declared in an internal stylesheet, an external stylesheet, and a web browser's default styles.
  • Internal stylesheet - An Internal stylesheet has the second highest priority. It will override styles declared in an external stylesheet and a web browser's default styles.
  • External stylesheet - An External stylesheet has the third highest priority. It will override a web browser's default styles.
  • Browsers default styles - A web browsers default styles have the lowest priority. A web browsers default styles are used when there is no style set for an element in an Inline, Internal, or External stylesheet.
